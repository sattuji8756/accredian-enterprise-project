# 🚀 Accredian Enterprise – Modern Landing Page

A fully responsive and modern **enterprise landing page** built using **Next.js 13 (App Router)** and **Tailwind CSS**. This project demonstrates scalable frontend architecture, clean UI design, and API integration.

---

## 📌 Project Overview

**Accredian Enterprise** is a professional business landing page designed to showcase services, clients, and user engagement features.

It includes:

* Hero Section
* Clients Section (Infinite Scroll)
* Features Section
* Testimonials
* FAQs
* Contact Form (API Integrated)

---

## 🛠️ Tech Stack

* ⚡ Next.js 13 (App Router)
* ⚛️ React.js
* 🎨 Tailwind CSS
* 📡 Next.js API Routes
* 📜 JavaScript (ES6+)
